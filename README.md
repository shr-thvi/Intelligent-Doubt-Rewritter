# Clarify

An AI-powered EdTech application that transforms unclear student doubts into clear, structured academic questions.

## Features
- Grammar correction
- Hinglish to English conversion
- Subject detection
- Follow-up questions
- Detailed explanations

## Installation

```bash
npm install
npm run dev

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`
